# messenger
![Alt text](/img/messenger.png)
- JAVA로 메신저를 구현하였습니다. 

# 동기
- Java와 3-Tier방식을 공부하기위해서 메신저를 제작하게 되었습니다<br>
- oracle DB연동에 대해서도 공부 및 연구를 위해 제작하게 되었습니다. <br> 

# 개발환경
> Back-End
- JDK 1.8
- Java 8 

> Front-End
- JavaFX
- Scene Builder

# 기능설명
- 회원가입<br>
- 로그인,로그아웃<br>
- ID,PW 찾기<br>
- 친구추가<br>
- 친구삭제<br>
- 친구찾기<br>
- 스레드풀을 이용하여 채팅 서버 구현 <br>
- p2p 대화하기<br>
